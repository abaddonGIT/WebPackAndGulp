## Requirements
- node >= 10.13.0
- yarn or npm

## Install
    yarn install    
#### Start dev server
    yarn start
#### Production build    
    yarn run build
#### Update package.json
    yarn run check
#### ESLint check
    yarn run lint
#### TS check
    yarn run ts-check
#### Fix code style with prettier
    yarn run fix-code-style
